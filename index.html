Bill Shupp's PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover shupp.github.com/pirum
Listing available packages: 
pear remote-list -c shupp
Installing a package: 
pear install shupp/package_name
Installing a specific version/stability: 
pear install shupp/package_name-1.0.0
pear install shupp/package_name-beta
Receiving updates via a feed: 
http://shupp.github.com/pirum//feed.xml

Deneb - Simple CRUD layer for DB based models

Deneb provides a consistent CRUD interface to using Zend_Db base models and model collections, as well as segregated read and write DB pools and selectors

  • Package name: Deneb
  • License: New BSD License
  • Maintainers: Bill Shupp (as lead)
  • Releases: 0.4.0 (alpha), 0.3.0 (alpha)
  • Install command: pear install shupp/Deneb

Predict - A partial port of the Gpredict program for satellite tracking

Predict is a partial PHP port of the Gpredict (http://gpredict.oz9aec.net/) program that allows real-time tracking and orbit prediction of satellites from two line element sets. It supports the SGP4 and SDP4 models for prediction.

  • Package name: Predict
  • License: GPL v2.1
  • Maintainers: Bill Shupp (as lead)
  • Releases: 0.1.2 (alpha), 0.1.1 (alpha), 0.1.0 (alpha)
  • Install command: pear install shupp/Predict

RandomString - A RandomString generator

Easy creation of random strings, limiting by lower case, upper case, numeric, and puncuation characters

  • Package name: RandomString
